Minha viso sobre Ruby

Gosto do Ruby por ter uma tima semntica e sintaxe.

Ruby esta se tornando uma linguagem bem madura a partir da verso 2.0 onde ocorreram vrias mudanas e uma das principais foram o aumento de sua performance.

Ruby composto por timos frameworks e bibliotecas como Rails e Chef. E tudo isso gerenciado por um sistema chamado Rubygems.

A comunidade Ruby esta cada dia mais forte e maior, sendo assim muito mais fcil achar solues e dicas.

Os programadores Ruby geralmente produzem solues elegantes.

Com essas caracterstica Ruby me convence que a minha linguagem favorita de "back-end".

Minha viso sobre Javascript

Javascript uma linguagem de programao de fcil acesso e que esta presente em quase todos lugares.

A baixa curva de aprendizado tem aumentado o nmero de interessados pela linguagem.

A linguagem tambm atraiu bons pensantes que desenvolveram timos frameworks e bibliotecas com o intuito de melhorar o jeito de se programar em Javascript, alguns deles so: jquery, backbone, ember, angular, marionette, grunt, gulp e entre outros mais.

Esse boom fez a comunidade se dispertar e implementar a verso 6 para o ECMA. Essa alterao ir trazer uma robustez enorme a linguagem que estava parada desde 2009 (quase 6 anos).

Javascript no esta presente s no front, atualmente temos cdigo Javascript rodando no server graas ao Node, e isto maravilhoso pois podemos ter Javascript dos dois lados tambm.
