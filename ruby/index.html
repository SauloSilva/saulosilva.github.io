ActiveModel Serializer no sinatra

Adaptando o active model serializer para o Sinatra.

ActiveModel Serializer no sinatra Atualmente o ActiveModelSerializer so fornece suporte para Rails. Neste post irei dar uma dica de como solucionar este problema para quando estiver utilizando o Sinatra. Esta solução é simples e rápida. Basta adicionar ao seu Gemfile a gem sinatra-active-model-serializers.
